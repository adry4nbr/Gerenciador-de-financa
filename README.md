💰 Gerenciador de Finanças PessoaisUm aplicativo web moderno e intuitivo para controle financeiro, desenvolvido com React + Vite e Tailwind CSS v4. O projeto permite o monitoramento de receitas e despesas com visualização de dados via gráficos e suporte total a temas claro e escuro.🚀 Funcionalidades Principais📦 Gestão de Transações (CRUD): Adicione, edite, visualize e exclua receitas e despesas.📊 Dashboard Inteligente: Saldo total e resumo de entradas/saídas do mês selecionado.📈 Gráficos Interativos: Visualização de gastos por categoria usando a biblioteca Recharts.📅 Filtro por Período: Navegação simples entre os meses do ano.🌗 Modo Escuro: Interface adaptável com troca dinâmica de temas e persistência de preferência.💾 Persistência de Dados: Uso de LocalStorage para manter seus dados salvos no navegador.🛠️ Tecnologias UtilizadasTecnologiaFunçãoReactBiblioteca principal (Interface)ViteFerramenta de build ultra-rápidaTailwind v4Estilização moderna e Dark ModeLucide ReactÍcones minimalistasRechartsGráficos e visualização de dadosUUIDGeração de IDs exclusivos📂 Estrutura do ProjetoPlaintextsrc/
